# FireMaze Plugin

A simple Minecraft plugin for Paper 1.21 that adds fire-based maze mechanics.

## Features

- **Heat damage over time** in designated areas
- **Lava trap triggers** that spawn fireballs
- **Fire mob spawn zones** that spawn blazes/blaze-like mobs
- **Timed flame wall sections** that shoot fireballs
- Configurable damage rates and cooldowns

## Commands

- `/firemaze reload` - Reload the configuration
- `/firemaze setheatdamage <damage>` - Set heat damage rate
